What is CSS ?
  1. CSS stands for Cascading style sheets

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media

  3. Css is not necessary CSS save a lot of work. It can control layout of multiple web pages all at once

  4. External style sheets are stored in CSS file

CSS can be added to HTML document in 3 ways :
  • Inline - by using the style attribute inside HTML element
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an External CSS file
  • The most common way to add CSS is to keep the styles in external CSS files.However , in this tutorial we will see inline and internal styles, because this is easier to demonstrate and easier for you to try it by yourself

When comparing CSS class vs ID the difference is that CSS class applies a style to multiple elements,ID ,on the other heading ,applies a style to one unique element ID is also special in that you can use a special URL to limk direclty to an element and it's used by Java script

In CSS , selectors are used to target a specific element or range of elements on a web page.Once an element has been targeted, a style or set of styles can be applied to the element

There is a wide range of selectors available . Two of the most commonly used are class and ID.Boh are usedmto target elements to which astyle should be applied.
